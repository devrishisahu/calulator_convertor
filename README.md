🧮 Calculator Converter & Snake Game

A multi-functional web application that combines a Calculator, Currency Converter, and the classic Snake Game into a single interactive platform. Built using pure HTML, CSS, and JavaScript, this project demonstrates core frontend development concepts including DOM manipulation, event handling, game logic, and API integration.
